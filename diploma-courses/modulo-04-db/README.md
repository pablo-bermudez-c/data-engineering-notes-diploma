# Módulo 04: Modelos y Gestión de Bases de Datos

Este módulo cubre los fundamentos del modelado y operación de bases de datos: desde el modelo relacional clásico hasta las familias NoSQL más relevantes para ingeniería de datos. La idea central es elegir y diseñar el motor correcto para cada problema.

## Requisitos previos

- Conocimientos básicos de SQL (Módulo 01, [Clase 3](../modulo-01-fundamentos/clase-03-consultas-sql-join-postgresql/README.md)).
- Estructuras de datos básicas (Módulo 03, [Clase 5](../modulo-03-estructuras/clase-05-grafos/README.md) para la clase de grafos).
