# Módulo 03: Estructuras de Datos y Algoritmos

Este módulo cubre los fundamentos de estructuras de datos y análisis de algoritmos, indispensables para escribir código eficiente sobre grandes volúmenes de datos. Vas a aprender no solo *qué* estructuras existen, sino *cuándo* usar cada una y *cómo* analizar el costo de tus soluciones.

## Requisitos previos

- Conocimientos básicos de Python (Módulo 02, clases 1 y 2): variables, tipos, condicionales, bucles y funciones.
