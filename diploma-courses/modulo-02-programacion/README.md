# Módulo 02: Programación en Python

Este módulo te introduce a Python como lenguaje principal del Data Engineer: desde la sintaxis básica hasta el manejo de archivos, librerías de cómputo numérico (NumPy), análisis tabular (Pandas) y consumo de APIs.

## Requisitos previos

- Conocimientos básicos del [Módulo 01: Fundamentos](../modulo-01-fundamentos/README.md) (terminal, entornos virtuales, SQL)
- Python 3.10+ instalado o acceso a Google Colab

| 2 | [Estructuras de Control y Funciones](clase-02-estructuras-y-funciones/README.md) | Condicionales, bucles, funciones, recursión y *scope* |
