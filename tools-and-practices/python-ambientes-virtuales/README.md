# Módulo: Python - Ambientes Virtuales

Aprende a crear y gestionar ambientes virtuales para tus proyectos de Python usando **uv**, una herramienta moderna que simplifica todo el proceso de configuración y manejo de dependencias.

## Requisitos previos

- Tener instalado [VS Code](https://code.visualstudio.com/)
- Saber usar la terminal a nivel básico (abrir, ejecutar comandos)
- Haber completado el módulo de [GitHub](../github/README.md) (recomendado)
