# Módulo: GitHub

Aprende a utilizar Git y GitHub para gestionar el código del equipo, colaborar de manera eficiente y mantener un historial ordenado de los cambios en nuestros proyectos.

## Requisitos previos

- Tener instalado [Git](https://git-scm.com/downloads)
- Tener una cuenta en [GitHub](https://github.com)
- Tener instalado [VS Code](https://code.visualstudio.com/)
